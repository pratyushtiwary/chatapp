# Chatten

## Installation

Run `npm install`

then to run development server run `npm run dev` on your cmd and goto [http://localhost:3000](http://localhost:3000)

## Libraries Used

- Crypto-js,
- Material-ui
- Axios,
- Framer-motion,
- Socket.io-client,
- Next, &
- React
